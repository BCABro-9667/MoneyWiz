# **App Name**: MoneyWiz

## Core Features:

- Dashboard Expense Creation: Allows users to quickly add new expenses via a simplified form on the dashboard.
- Expense Management Page: Dedicated page for managing a specific expense, including adding expenditures, viewing balance, and editing details.
- Real-time Balance Updates: Automatically updates the current balance in real-time when expenditures are added or modified.
- Expenditure Management: Enables adding, editing, and deleting expenditures within an expense.
- Expenditure Filtering: Search bar functionality to filter expenditures by name or amount.
- Print View Export: Generates a printer-friendly view of the expense details and expenditures.
- AI-Powered Insight Tool: Analyze user spending habits and provide summaries on most frequent expenditure categories and insights to identify areas for potential savings.

## Style Guidelines:

- Primary color: Forest green (#78C841) to create a sense of trustworthiness, aligned to the theme specified in the request.
- Background color: Very light green (#F0F7EC) to complement the primary.
- Accent color: Sky blue (#41B6C8) for interactive elements; a good contrast against the primary color and provides an engaging feel.
- Body and headline font: 'PT Sans', a sans-serif font for a clean, modern, readable style suitable for an app's user interface.
- Cards and forms should use curved design with a 50px border radius, and be visually distinct with soft shadows. Content should be well-spaced.
- Use subtle transitions on hover states to enhance interactivity, making it more responsive and engaging.
- Use a consistent set of icons that match the green color scheme, ensuring they are modern, recognizable, and add clarity.
